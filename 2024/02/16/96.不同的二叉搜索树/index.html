动态规划之不同的二叉搜索树

发表于2024-02-16|更新于2024-08-09|算法动态规划
|字数总计:1k|阅读时长:3分钟|阅读量:

力扣题目链接

题目描述

给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

示例

示例1

输入：n = 3
输出：5

示例2

输入：n = 1
输出：1

思路

这道题刚开始看有点懵，不容易想到用动态规划来求解，首先树结构就是由节点的重复来组成的，比如第一层节点再填一层就是两层，两层再填一层就是三层，还是可以找到重复的逻辑的。
当n=1的时候，只有一种情况

当n=2的时候，有两种情况

当n=3的时候，从上面的图示中也可以看到有5种情况。

那么它们之间有什么规律呢？
可以看到当1为根节点时，它有两种情况，正好对应n=2的时候（不用考虑具体的节点值）
当2为根节点时，它有一种情况，对应n=1的时候
当3为根节点时，它有两种情况，同样对应n=2的时候
从这里可以看到一些重叠问题了，也就是dp[1]和dp[2]可以推导出dp[3]
那么具体怎么导呢？
首先n=3的情况dp[3] = 根节点为1的个数+根节点为2的个数+根节点为3的个数
根节点为1的个数= 右子树有两个元素的搜索树数量左子树有零个元素的搜索树数量
根节点为2的个数= 右子树有一个元素的搜索树数量左子树有一个元素的搜索树数量
根节点为3的个数= 右子树有零个元素的搜索树数量左子树有两个元素的搜索树数量
有两个元素的搜索树数量为dp[2],有一个元素的搜索树数量为dp[1]
dp[3] = dp[2]dp[0]+dp[1]dp[1]+dp[0]dp[2]

确定状态数组dp

状态数组可以用一个一维数组来表示，dp[i]表示元素个数为i的不同二叉搜索树的个数

确定递推方程

之前已经推导的差不多了，dp[i] += dp[j-1]*dp[i-j]，其中j表示根节点的元素值，比如对于n=3来说，j就可以是1、2、3。j-1就是以j为根节点的左子树节点个数，i-j就是以j为根节点的右子树节点个数

确定状态数组初始值

对于0个节点来说，即空树，给它赋值dp[0]=1，否则后续的乘积计算都为0，对于1个节点来说，dp[1]=1,

确定遍历顺序

同理，从前往后遍历，需要知道两层遍历分别需要确定的值，对于外层，它的一次循环会确定一个dp[i]，。对于里层循环，是具体来i进行讨论的，比如i有几种情况划分啊，j是从1-i,这么多种情况来划分这i个元素组成的二叉搜索树

举例推导dp数组

其实之前已经推到过n=3的情况了

i 0 1 2 3
dp[i] 1 1 2 5
